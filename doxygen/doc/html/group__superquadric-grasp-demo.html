Description

This module provides a wrapper code for performing the superquadric modeling and grasping demo. A tutorial on how to use the module is provided here: tutorial.

This page illustrates the main parameters of the module (Section Parameters), the available ports and services.

Parameters

  • –from: Configuration file name.
  • –objname: The name of the object to be modeled and grasped.
  • –hand_for_computing: Select the hand for which we want to compute a grasping pose. Available values: right, left, both.
  • –hand_for_moving: Select the hand for which we want to use for executing the grasping. Available values: right, left.
  • –choose_hand: Select automatically the best hand for grasping the object. Available values: true, false.
  • –filtered: Filter multiple superquadrics in order to improve the object model. Available values: on, off.
  • –n_pc: Number of point clouds used for averaging the superquadrics.

Input Ports

  • /superquadric-grasp-demo/img:i [yarp::sig::ImageOfPixelRgb] [default carrier:tcp]: Receive the image from the left camera for saving the colored point cloud.
